# TP02_2C_Bernardi_CarGame

CarShooter es un juego de acción y conducción en 3D desarrollado en Unity, donde controlás un auto armado preparado para sobrevivir en pistas llenas de peligros.

Tu objetivo es recorrer distintas pistas, eliminar enemigos y conseguir la mayor cantidad de score posible mientras evitás minas explosivas y aprovechás rampas para ganar ventaja.

Manejate a toda velocidad, cambiá de arma y destruí todo a tu paso.

## 🕹️ Controles

| Acción | Tecla / Botón |
|--------|----------------|
| Mover auto | `W / A / S / D` |
| Frenar | `Espacio` |
| Apuntar | `Mouse` |
| Disparar | `Click izquierdo` |
| Cambiar arma | `Q` |
| Pausa | `Esc` |

## ⚔️ Mecánicas Principales

🚗 **Sistema de conducción:** control completo del vehículo con aceleración, giro y frenado.

🔫 **Sistema de combate:**
- Diferentes armas para eliminar enemigos.
- Cambio rápido de arma durante la partida.

💣 **Obstáculos y peligros:**
- Minas explosivas distribuidas por la pista.
- Rampas para realizar saltos y tomar ventaja.

🏎️ **Variedad de contenido:**
- Distintas pistas para recorrer.
- Diferentes autos jugables.

📊 **Sistema de puntuación:**
- Eliminar enemigos suma score.
- Sobrevivir y mantener el control mejora tu rendimiento.

🏁 **Objetivo del juego:**
- Conseguir la mayor cantidad de puntos posible derrotando enemigos y recorriendo las pistas.
